# Proyectos de JavaScript

## Descripción
Este repositorio incluye varias interfaces que he desarrollado a lo largo de mi carrera utilizando esta tecnología o lenguaje.
El objetivo de estos proyectos es aplicar los conocimientos teóricos, como la Programación Orientada a Objetos (POO) y la logica de programacion a la práctica, realizando operaciones matematicas basicas, ejercicios comparativos, validaciones, entre otros. 
Si bien son ejercicios muy sencillos pero que me ayudaron a afianzar los conocimientos basicos y a aplicar la logica en la programacion.

## Funcionalidades Implementadas
Estos proyectos incluyen las siguientes funcionalidades:
-  Estructura de control de flujo o condicionales (if, else).
-  Bucles iterativos o de repeticion (for, while, do while,switch).
-  Operadores logicos (!, <, >, ==, ).
-  Operadores aritmeticos (+, -, *, /).

## Aprendizajes Personales
Al trabajar en estos proyectos, pude aprender y mejorar en varias áreas:
-  Aprendí a aplicar principios fundamentales de la Programación Orientada a Objetos.
-  Aprendí a trabajar con la logica de programacion y a aplicar ciertos operadores, bucles y demas segun corresponda.
-  Desarrollé la capacidad de gestionar proyectos de software de manera eficiente.
  
Este proyecto me permitió mejorar mis habilidades en backend, así como también en resolución de problemas.
